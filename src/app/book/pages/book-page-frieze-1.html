Frieze Groups



The Frieze Groups create patterns that repeat infinitely in opposite directions.


A page does not do a pattern justice. It cuts them off when really they continue repeating forever.



This repetition introduces another type of symmetry - these patterns can shift over and still be the same pattern.



This shift is called translation.
